# Set up Controllers for BE

1. Click vào thư mục controllers → add → Controller

![image.png](Set%20up%20Controllers%20for%20BE/image.png)

1. Chọn API → API Controller with actions, using Enity Framwork → Add

![image.png](Set%20up%20Controllers%20for%20BE/image%201.png)

1. Chọn ModelClass and DbContextClass như trong hình (giữ nguyên Controller name → Add

![image.png](Set%20up%20Controllers%20for%20BE/image%202.png)

1. Chờ đợi code sinh ra trong Controller 

> **(Lưu ý chỉ có các tác vụ cơ bản CRUD, các tính năng nâng cao, cần tự ghi vào**
